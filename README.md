## React Boilerplate

It is a simple React project boilerplate, it uses Webpack, Babel, React Router and PreCSS.

### Usage

- `git clone https://github.com/AbeEstrada/react-boilerplate.git`
- `cd react-boilerplate`
- `npm install`
- `npm start`
- Go to `http://localhost:8080/`

### Build

- `npm run build`
