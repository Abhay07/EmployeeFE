## React Boilerplate

It is a basic React project boilerplate, it uses Webpack, Babel, React Router and SASS.

### Usage [DEV]

- `git clone https://github.com/AbeEstrada/react-boilerplate.git`
- `cd react-boilerplate`
- `npm install`
- `npm start`
- Open your browser: `http://localhost:8080/`

### Build

- `npm run build`
